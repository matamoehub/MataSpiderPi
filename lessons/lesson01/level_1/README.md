# Lesson 1 - Moving Around

## Goal
Learn to move SpiderPi safely and confidently.

## What you will use
- `from student_robot_v2 import bot`
- `bot.body` for walking and turning
- `bot.sound` or `bot.speech` if you want the robot to talk while it moves

## Success criteria
- You can move forward, backward, left, right, and turn.
- You can build a short movement sequence.
- You can stop the robot safely when you finish.
