# Lesson 6 - Tracking and Tags

## Goal
Connect vision data to robot behaviour.

## What you will use
- `bot.move` for body movement
- `bot.arm` for the SpiderPi gripper arm
- `bot.actions` for action groups
- `bot.vision` for camera-guided tasks
- `bot.sonar` and `bot.camera` for sensing

## Success criteria
- You can explain which SpiderPi subsystem you used.
- You can run at least one movement task and one sensing or arm task.
- You can extend the notebook with your own creative challenge.
