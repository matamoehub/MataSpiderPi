# Lesson 13 - Rock Paper Scissors Vision

## Goal
Use SpiderPi's camera and MediaPipe hand recognition to play rock, paper, scissors.

## What you will use
- `bot.vision.recognize_hands()`
- `bot.display`
- `bot.lights`
- `bot.speech`
- `bot.body`

## Success criteria
- You can capture one camera frame and inspect the detected hand gesture.
- You can map the gesture to `rock`, `paper`, or `scissors`.
- You can make SpiderPi react when it wins, loses, or draws.

## Teaching note
This lesson uses the MataSpiderPi V2 call style so it matches MataTurboPi naming.
